Variantes de layout

Para um mesmo layout, crie dois frames com nomes no padrão:

Nome do frameUso
layout/01Padrão — fonte maior, textos curtos
layout/01/+textoVariante — fonte menor, textos longos

A ferramenta detecta automaticamente e exibe o toggle "Fonte do Titulo Maior | Titulo para mais palavras" acima do canvas. Ao trocar, o texto já digitado é preservado.

Fontes comerciais

Coloque os arquivos em:
fonts/{NomeDaFonte}/400.woff2
Pesos suportados: 300, 400, 600, 700.
Nome EXATO igual ao retornado pelo Figma.

Observações

  • Cada camada é exportada como PNG individual — blend modes entre camadas podem diferir
  • maskFoto define o contorno de clip da foto com precisão
  • TEXT invisível ainda é lido pela API → posição, fonte e cor extraídos corretamente
  • blocoGlass: blur e cor do fill lidos diretamente do Figma
